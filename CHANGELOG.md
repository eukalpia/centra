# Changelog

## 0.1.0

- Added first-run Cinder setup wizard with keyboard and mouse support.
- Added ten interface languages.
- Added local, SSH, Docker container, Docker image, and Docker Compose sources.
- Added explicit multi-algorithm registry with legacy/obsolete warnings.
- Added external custom hash command support.
- Added project detection and editable exclusion policies.
- Added canonical JSON and compatibility text manifests.
- Added password-protected ZIP packaging.
- Added manifest comparison and source verification.
- Added Ed25519 key generation, signing, and verification.
- Added CLI automation, stable exit codes, diagnostics, tests, and release workflows.
