# Persistent settings

Centra stores the selected interface language after the first completed setup. Existing installations with profiles also skip repeated language selection automatically.

The global settings screen is available from the dashboard at any time and controls interface language, appearance, destructive-action confirmation, root-scan confirmation, and the default verification mode.

Profile-specific scan policies remain separate from global settings. They include full or fast verification, read-error handling, retries, timeouts, size and count limits, directory depth, symbolic-link behavior, one-filesystem best effort, and trusted baseline files.
